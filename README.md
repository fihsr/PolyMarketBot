# PolyMarketBot
Telegram Bot PolyMarket
Телеграм бот для торговли на Polymarket, может искать рынки, ставить ордера и тд
Основные функции:
🔐 Authenticate
📊 Browse Markets
📈 Market Statistics
🎯 Select Outcome
📈 Analyze Order Book
💰 Check Balance
⚡ Place Market Order
📝 Place Limit Order
📋 View Open Orders
❌ Cancel Orders
📊 Track Price
📊 My Positions
🔄 Refresh
Бот работает полностью испрвно, весь код в одном файле, ваам надо заменить только TOKEN бота
По желанию такой проект вы можете превратить в огромный проект по копи трейдингу или использовать так.
